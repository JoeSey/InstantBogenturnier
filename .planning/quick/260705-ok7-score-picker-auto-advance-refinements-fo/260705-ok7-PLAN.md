<objective>
Follow-up to quick task 260705-lpv, which added a naive cross-shooter auto-advance to the score-entry tap-button picker. This task retires the cross-row jump and adds two more refinements:

1. Editing an already-filled cell must close the dialog after the pick — never auto-advance, since the trainer was correcting a single value, not filling a row.
2. Auto-advance must stay within the SAME archer's row only. It must never jump to the next archer's row anymore; once the current row has no more empty arrows, the dialog closes.
3. The dialog title must show a live per-arrow preview of the current row as it fills in, e.g. "Punkte von Anna (8 - -)" (dash = not yet entered), updating after every pick — including the just-made pick, ahead of the async liveQuery refresh.

Purpose: Match the trainer's actual mental model at the range — auto-advance is for filling one archer's row quickly, not for hopping between archers, and correcting a mistake shouldn't drag them into an unrelated row. The live preview gives instant confirmation of what's been entered so far without closing the dialog to check the table.
Output: A same-row-only `findNextEmptyArrowInRow` pure helper (replacing the retired cross-row `findNextEmptyArrow`), an updated `ScorePicker.svelte` with a `rowPreview` prop, a 2-arg `pickerTitle` string function, and `ScoreEntry.svelte` wiring that tracks whether the tapped cell was already filled and accumulates just-picked values for the live preview.
</objective>

function handleScoreSelect(value: ScoreValue) {
  if (!pickerCell || !roundsConfig) return;
  const { shooterId, arrowIndex } = pickerCell;
  db.scores.put({ shooterId, roundIndex: selectedRound, passeIndex: selectedPasse, arrowIndex, value, finalized: false }).catch(...);

  const justPickedKey = `${shooterId}-${arrowIndex}`;
  const isFilled = (sId: number, aIdx: number) => {
    const key = `${sId}-${aIdx}`;
    return key === justPickedKey || currentPasseScoreByKey.has(key);
  };
  pickerCell = findNextEmptyArrow(rows, roundsConfig.arrowsPerPasse, shooterId, arrowIndex, isFilled);
}

function cancelPicker() {
  pickerCell = null;
}
