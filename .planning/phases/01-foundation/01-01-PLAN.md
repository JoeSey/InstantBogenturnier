## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| npm registry → build | Third-party packages enter the bundle at install/build time (supply chain). |
| static host → browser | Precached assets served to the installed PWA; no server-side trust decisions, no network calls at runtime. |
| browser → localStorage/IndexedDB | Local-only, single-operator, single-device persistence; no cross-user or cross-origin trust. |
