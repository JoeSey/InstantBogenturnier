    Implement `buildScoresheetPdfDoc(roundsConfig: RoundConfig, settings: Pick<SettingsRecord, 'title' | 'logoLeftBlob' | 'logoRightBlob'> | undefined): Promise<jsPDF>`:
    - `new jsPDF({ orientation: 'portrait', unit: 'mm', format: 'a5' })` — A5 portrait per
      SHEET-05 (this is the one deliberate format difference from pdfExport.ts/certificateExport.ts,
      which both use `format: 'a4'`).
    - Render the header exactly like `buildResultsPdfDoc`'s header block (lines 84-116 of
      pdfExport.ts): left logo via `containFit` scaled into a smaller box appropriate for A5
      (A5 is roughly 148x210mm vs A4's 210x297mm, so scale `LOGO_MAX_WIDTH`/`LOGO_MAX_HEIGHT`
      down proportionally, e.g. 18mm/14mm instead of pdfExport.ts's 25mm/20mm), right logo
      mirrored to the page-width minus margin, centered title text via `settings.title` if
      present. Use `doc.internal.pageSize.getWidth()` for centering (A5 width in 'mm' unit is
      148mm, not a hardcoded literal — always read from `doc.internal.pageSize`).
    - Render 4 blank handwriting fields below the header (SHEET-03): "Name:", "Klasse:",
      "Schießplatz:" (shooting line), "Schreiber:" — each as a label plus a blank underline
      (`doc.line(x1, y, x2, y)`) for handwriting, laid out in a 2x2 grid or single column
      depending on available width — pick whichever avoids horizontal overflow on A5's 148mm
      width with 15-20mm margins.
    - Render the grid (SHEET-02): `roundsConfig.numberOfRounds` rounds, each containing
      `roundsConfig.passesPerRound` passes, each containing `roundsConfig.arrowsPerPasse` arrow
      cells, drawn with `doc.rect()` per cell in a compact table layout with round/pass labels
      (e.g. "R1", "P1") on the left. Compute cell height dynamically so the full grid fits
      within the remaining A5 vertical space after the header/fields and before the footer
      signature lines — do not let it overflow to a second page (verified by the
      `getNumberOfPages() === 1` test). If necessary, arrange multiple passes side-by-side
      per row to keep the total height within budget for larger configs (e.g. numberOfRounds=2,
      passesPerRound=5).
    - Render 2 blank signature lines at the bottom (SHEET-06): "Unterschrift Schütze" and
      "Unterschrift Schreiber", each as a `doc.line()` with the label centered/left-aligned
      beneath it, matching official tournament scoresheet conventions.
    - Return `doc`.
